## 🚀설치 및 실행 방법

### 설치

1. **저장소 복제하기**

```
git clone https://github.com/LeeJuHwan777/Xcoding.git
```

2. **루트 디렉토리에 대한 종속성 설치**

```
// 루트 디렉토리
npm install
```

### 실행
```
npm run dev
```
